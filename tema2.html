Tema 02

Grundlæggende web

I tema 02 blev vi for første gang introduceret til HTML og CSS, grundstenene i webudvikling, ved hjælp af VS Code. Det var vores første skridt ind i den kreative og tekniske verden af hjemmesidedesign. Opgaven var at skabe vores allerførste hjemmeside, og for at hjælpe os i gang fik vi udleveret billeder og wireframes som skabeloner fra undervisningen. Det var en spændende oplevelse at se, hvordan de linjer af kode, vi skrev, blev forvandlet til en visuel og funktionel hjemmeside. Fra at forstå strukturen med HTML til at style indholdet med CSS lærte vi, hvordan man kunne bringe idéer til live på skærmen.
